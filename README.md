# Airbnb Backend Clone

Backend system inspired by Airbnb built using Spring Boot.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Hibernate
- REST APIs
- Maven
- Git & GitHub

## Features Implemented
- Hotel Management APIs
- DTO Mapping
- Service Layer Architecture
- Global Exception Handling
- Standard API Response Wrapper

## In Progress
- Booking System
- Authentication & Authorization
- Payment Integration
- Admin APIs
